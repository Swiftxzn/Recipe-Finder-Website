# 🍃 Healthy Recipe Finder

Projeto desenvolvido como atividade avaliativa, com foco em criação de interface moderna utilizando **React + Vite + TailwindCSS**.

---

## 📌 Sobre o projeto

O **Healthy Recipe Finder** é um site que apresenta receitas rápidas e saudáveis, com um layout moderno e responsivo.

A ideia é facilitar a escolha de refeições simples, sem complicação e com ingredientes acessíveis.

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 TailwindCSS
* 🔀 React Router

---

## 🎯 Funcionalidades

* Header com navegação entre páginas
* Seção principal com destaque (Hero)
* Seção de benefícios ("What you'll get")
* Layout responsivo
* Seção informativa ("Built for real life")
* Call-to-action com botão de navegação
* Estilização moderna com Tailwind

---


---

## 🛠️ Como rodar o projeto

```bash
# instalar dependências
npm install

# rodar o projeto
npm run dev
```
